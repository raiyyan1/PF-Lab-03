# CL1002 Programming Fundamentals - Lab
**Raiyyan Hussain**\
**25K-6520**
### Objectives:
- Introduce students to GitHub and setting up their profile with their first repository.
- Introduce the basic syntax and structure of the C programming language.
- Teach students how to write and compile simple C programs.
- Familiarize students with fundamental C programming concepts, such as variables and data types.
- Develop problem-solving skills by writing C programs to solve basic computational problems.
